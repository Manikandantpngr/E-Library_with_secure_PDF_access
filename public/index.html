E-Library — Secure PDF Access

A minimal demo server that protects downloads and applies a per-download watermark.

Quick actions

  • Register: POST /auth/register
  • Login: POST /auth/login
  • Upload book (PDF): POST /upload/book (multipart)
  • List books: GET /books
  • Download (protected): GET /download/:id

Example: after login, call:

curl -H "Authorization: Bearer <token>" http://localhost:3000/download/<book_id> --output book.pdf

Notes

  • Set environment variables in .env (see .env.example).
  • Server runs on port 3000 by default.
If the page is still blank after starting the server, check the server logs.
